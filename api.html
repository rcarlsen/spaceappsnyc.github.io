twilio

Supernova sponsor twilio makes using messy telecom hardware easy and exposes a globally available cloud API that developers can interact with to build scalable, intelligent, and complex communications systems. Their products allow you to make voice driven applications, SMS, MMS, and short code messaging systems, and much more.

http://www.twilio.com/api

SparkPost

Supernova sponsor SparkPost is the transactional email solution designed specifically for developers, based on the Momentum platform from Message Systems. It provides the same deliverability, scalability and speed that Message Systems provides to the biggest senders in the world, in an easily integrated, pay-as-you-go cloud solution.

https://www.sparkpost.com/features
https://www.sparkpost.com/docs/getting-started-0
